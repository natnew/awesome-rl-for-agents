# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

All participants — contributors, maintainers, and anyone interacting with the project — are
expected to uphold its standards. In short: be respectful and constructive, assume good faith, and
keep discussion focused on the resources and the list.

To report unacceptable behaviour, open an issue or contact the maintainers via the
[repository](https://github.com/natnew/awesome-rl-for-agents). Reports will be reviewed and handled
in line with the Contributor Covenant's enforcement guidelines.
